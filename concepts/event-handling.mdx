## Overview

yolkbot uses an event-driven architecture. The Bot class emits events for game actions, and you can listen to these events to react to game state changes.

## Event System Architecture

The Bot class implements a custom event system with two types of listeners:

```javascript Event system from src/bot.js:998-1023
// Register event listener
bot.on('eventName', (arg1, arg2) => {
  console.log('Event fired!', arg1, arg2);
});

// Listen to ALL events
bot.onAny((eventName, ...args) => {
  console.log(`Event ${eventName} fired with args:`, args);
});

// Remove event listener
bot.off('eventName', callback);

// Remove all listeners for an event
bot.off('eventName');
```

### Internal Event Emission

```javascript Event emission from src/bot.js:1012-1017
// Events are emitted internally by the bot
bot.$emit('eventName', arg1, arg2, arg3);

// This triggers:
// 1. All specific listeners registered with bot.on('eventName', ...)
// 2. All global listeners registered with bot.onAny(...)
```

<Info>
  The `$emit` method is internal. You should only use `bot.on()` and `bot.onAny()` to listen to events, not emit them yourself.
</Info>

## Event Listener Patterns

### Single Event Listener

```javascript
const handleChat = (player, message, flags) => {
  console.log(`${player.name}: ${message}`);
};

bot.on('chat', handleChat);

// Later, remove listener
bot.off('chat', handleChat);
```

### Multiple Event Listeners

```javascript
bot.on('playerDeath', (killed, killer) => {
  console.log('Death handler 1');
});

bot.on('playerDeath', (killed, killer) => {
  console.log('Death handler 2');
});

// Both handlers will be called
```

### Global Event Listener

```javascript Global listener from src/bot.js:1003-1005
bot.onAny((eventName, ...args) => {
  console.log(`[${eventName}]`, ...args);
});

// Logs ALL events
```

### Conditional Event Handling

```javascript
bot.on('playerDeath', (killed, killer) => {
  if (killed.id === bot.me.id) {
    console.log('I died!');
    // Respawn logic
  } else if (killer?.id === bot.me.id) {
    console.log('I got a kill!');
  }
});
```

### Event Chaining

```javascript
let lastAttacker = null;

bot.on('playerDamage', (victim, attacker) => {
  if (victim.id === bot.me.id) {
    lastAttacker = attacker;
  }
});

bot.on('playerDeath', (killed, killer) => {
  if (killed.id === bot.me.id && killer) {
    console.log('Killed by:', killer.name);
    console.log('Last attacker was:', lastAttacker?.name);
  }
});
```

## Message Handling Patterns

### Processing Game Messages

The Bot processes incoming WebSocket messages automatically:

```javascript Message processing from src/bot.js:1093-1119
bot.game.socket.onmessage = (msg) => bot.processPacket(msg.data);

// Internal packet processing:
// 1. Initializes CommIn buffer with packet data
// 2. Reads command codes from buffer
// 3. Finds handler for each command
// 4. Handler updates bot state and emits events
// 5. Continues until buffer is empty
```

<Info>
  You don't need to manually process packets. The Bot handles all message parsing and emits high-level events.
</Info>

### Packet Structure

Packets contain one or more commands:

```javascript Packet handler structure from src/bot.js:1044
bot.packetHandlers = {
  [CommCode.syncThem]: processSyncThemPacket,
  [CommCode.fire]: processFirePacket,
  [CommCode.hitThem]: processHitThemPacket,
  [CommCode.chat]: processChatPacket,
  [CommCode.addPlayer]: processAddPlayerPacket,
  [CommCode.die]: processDiePacket,
  // ... 30+ more handlers
};
```

Each handler:
1. Reads data from `CommIn` buffer
2. Updates bot state (players, game, etc.)
3. Emits relevant events

### Example: Chat Packet Handler

```javascript Chat handler from src/packets/chat.js
const processChatPacket = (bot) => {
  const id = CommIn.unPackInt8U();
  const msgFlags = CommIn.unPackInt8U();
  const text = CommIn.unPackString();

  const player = bot.players[id];
  if (player) bot.$emit('chat', player, text, msgFlags);
};
```

## Complete Event Handling Example

```javascript
import { Bot, Intents, Movement } from 'yolkbot';

const bot = new Bot({
  intents: [
    Intents.PLAYER_HEALTH,
    Intents.PATHFINDING,
    Intents.PING
  ]
});

// Authentication events
bot.on('authSuccess', (account) => {
  console.log('Logged in! Egg balance:', account.eggBalance);
});

// Game connection events
bot.on('gameReady', () => {
  console.log(`Joined ${bot.game.gameMode} on ${bot.game.map.name}`);
});

// Player events
bot.on('playerJoin', (player) => {
  console.log(`${player.name} joined`);
});

bot.on('botJoin', (me) => {
  console.log('I am player', me.id, 'on team', me.team);
});

// Combat events
bot.on('playerDeath', (killed, killer) => {
  if (killed.id === bot.me.id) {
    console.log('I died to', killer?.name || 'unknown');
  } else if (killer?.id === bot.me.id) {
    console.log('I killed', killed.name, '- streak:', bot.me.streak);
  }
});

bot.on('playerDamage', (victim, attacker, damage) => {
  if (attacker?.id === bot.me.id) {
    console.log('Hit', victim.name, 'for', damage);
  }
});

// Chat events
bot.on('chat', (player, message) => {
  console.log(`[CHAT] ${player.name}: ${message}`);
  
  if (message.toLowerCase().includes('hello')) {
    bot.emit('chat', 'Hello!');
  }
});

// Respawn events
bot.on('playerRespawn', (player) => {
  if (player.id === bot.me.id) {
    console.log('Respawned at', player.position);
    bot.emit('move', Movement.Forward);  // Start moving
  }
});

// Tick events
let tickCount = 0;
bot.on('tick', () => {
  tickCount++;
  
  if (tickCount % 30 === 0) {  // Every 30 ticks (~1 second)
    const target = bot.getBestTarget();
    if (target && bot.canSee(target)) {
      bot.emit('lookAt', target.id);
      bot.emit('fire', true);
    }
  }
});

// Global event logger
bot.onAny((eventName, ...args) => {
  console.log(`[EVENT] ${eventName}`);
});

// Start the bot
await bot.loginAnonymously();
const game = await bot.findPublicGame('na-1', 'ffa');
await bot.join('EventBot', game);
```

## Best Practices

<AccordionGroup>
  <Accordion title="Always check if player exists">
    ```javascript
    bot.on('playerDeath', (killed, killer) => {
      // killer might be null if player fell off map or disconnected
      if (killer) {
        console.log(`${killed.name} killed by ${killer.name}`);
      } else {
        console.log(`${killed.name} died`);
      }
    });
    ```
  </Accordion>

  <Accordion title="Use specific events over global listeners">
    ```javascript
    // Preferred - specific and efficient
    bot.on('playerDeath', handler);
    
    // Avoid for production - processes ALL events
    bot.onAny((event, ...args) => {
      if (event === 'playerDeath') handler(...args);
    });
    ```
  </Accordion>

  <Accordion title="Clean up listeners when done">
    ```javascript
    const chatHandler = (player, message) => {
      console.log(message);
    };
    
    bot.on('chat', chatHandler);
    
    // Later...
    bot.off('chat', chatHandler);  // Remove specific handler
    // or
    bot.off('chat');  // Remove ALL chat handlers
    ```
  </Accordion>

  <Accordion title="Don't block event handlers">
    ```javascript
    // Bad - blocks event loop
    bot.on('playerDeath', async (killed, killer) => {
      await longRunningOperation();  // Other events can't fire!
    });
    
    // Good - handle async work separately
    bot.on('playerDeath', (killed, killer) => {
      longRunningOperation().catch(console.error);
    });
    ```
  </Accordion>
</AccordionGroup>
