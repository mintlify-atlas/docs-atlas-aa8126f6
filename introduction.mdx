## What is yolkbot?

yolkbot is a fast, optimized, and easy-to-use Shell Shockers automation library. Built for both Node.js and browser environments, yolkbot provides everything you need to create sophisticated bots for the popular online game Shell Shockers.

Whether you're building a simple bot that joins games or a complex AI with pathfinding and combat logic, yolkbot handles the low-level WebSocket communication, game state management, and protocol details so you can focus on bot behavior.

## Quick example

Create a bot that joins a game in just a few lines:

```javascript
import { Bot } from 'yolkbot';

const bot = new Bot();

await bot.loginAnonymously();
const game = await bot.findPublicGame('us-east', 'ffa');
await bot.join('MyBot', game);

bot.on('gameJoined', () => {
  console.log('Bot has joined the game!');
  bot.emit('spawn');
});
```
