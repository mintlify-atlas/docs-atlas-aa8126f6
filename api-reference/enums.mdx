Enums provide error codes, intent flags, and other enumerated values used throughout yolkbot.

## Import

```javascript
import { 
  APIError,
  LoginError,
  GameJoinError,
  Intents,
  CleanupLevel
} from 'yolkbot';
```

## Error Enums

### APIError

Base API error codes.

| Error | Value | Description |
|-------|-------|-------------|
| `WebSocketConnectFail` | `'websocket_connect_fail'` | Failed to connect to WebSocket |
| `MissingParams` | `'missing_params'` | Required parameters missing |
| `NetworkFail` | `'firebase_network_failed'` | Firebase network error |
| `ServicesClosedEarly` | `'services_closed_early'` | Services shut down unexpectedly |
| `InternalError` | `'unknown_error'` | Unknown internal error |
| `FirebaseRateLimited` | `'firebase_rate_limited'` | Firebase rate limit hit |

```javascript
try {
  await bot.someMethod();
} catch (error) {
  if (error.code === APIError.NetworkFail) {
    console.log('Network connection failed');
  }
}
```

### LoginError

Login-specific error codes (extends APIError).

| Error | Value | Description |
|-------|-------|-------------|
| All from APIError | - | Inherits all APIError codes |
| `AccountBanned` | `'account_banned'` | Account is banned |

```javascript
try {
  await bot.login(email, password);
} catch (error) {
  if (error.code === LoginError.AccountBanned) {
    console.log('This account has been banned');
  } else if (error.code === LoginError.NetworkFail) {
    console.log('Login failed due to network issues');
  }
}
```

## Error Handling Patterns

### Basic Try-Catch

```javascript
try {
  await bot.login(email, password);
} catch (error) {
  console.error('Login failed:', error.code, error.message);
}
```

### Specific Error Handling

```javascript
try {
  await bot.joinGame(GameMode.Team, PlayType.JoinPublic);
} catch (error) {
  switch (error.code) {
    case GameJoinError.GameNotFound:
      console.log('Game not found, retrying...');
      await bot.joinGame(GameMode.Team, PlayType.JoinPublic);
      break;
    case GameJoinError.WebSocketConnectFail:
      console.log('Connection failed, check network');
      break;
    default:
      console.error('Unknown error:', error.code);
  }
}
```

### Session Expiration

```javascript
try {
  await bot.buyItem('hat_123');
} catch (error) {
  if (error.code === BuyItemError.SessionExpired) {
    console.log('Session expired, logging in again...');
    await bot.login(email, password);
    await bot.buyItem('hat_123');
  }
}
```

### Retry Logic

```javascript
async function redeemWithRetry(code, maxAttempts = 3) {
  for (let i = 0; i < maxAttempts; i++) {
    try {
      return await bot.redeemCode(code);
    } catch (error) {
      if (error.code === RedeemCodeError.Ratelimited) {
        console.log(`Rate limited, waiting... (attempt ${i + 1})`);
        await new Promise(r => setTimeout(r, 5000));
      } else {
        throw error; // Don't retry on other errors
      }
    }
  }
  throw new Error('Max retry attempts reached');
}
```
