title: Constants
description: Game constants and configuration values for Shell Shockers
---

The constants module provides static game configuration values, including game modes, movement flags, weapon data, close codes, and more.

## Import

```javascript
import { 
  GameMode, 
  Movement, 
  PlayType, 
  Team,
  CloseCode,
  GunList
} from 'yolkbot';
```

## Game Configuration

### GameMode

Available game modes in Shell Shockers.

| Constant | Value | Description |
|----------|-------|-------------|
| `FFA` | 0 | Free For All |
| `Team` | 1 | Team Deathmatch |
| `Spatula` | 2 | Capture the Spatula |
| `KOTC` | 3 | King of the Coop |

```javascript
const bot = new Bot();
await bot.login(email, password);
await bot.joinGame(GameMode.Team, PlayType.JoinPublic);
```

### PlayType

Game join types.

| Constant | Value | Description |
|----------|-------|-------------|
| `JoinPublic` | 0 | Join a public match |
| `CreatePrivate` | 1 | Create a private lobby |
| `JoinPrivate` | 2 | Join a private lobby |

```javascript
// Join a public FFA game
await bot.joinGame(GameMode.FFA, PlayType.JoinPublic);

// Create a private KOTC lobby
await bot.joinGame(GameMode.KOTC, PlayType.CreatePrivate);
```

### Movement

Movement and control flags used for player input.

| Constant | Value | Description |
|----------|-------|-------------|
| `Forward` | 1 | Move forward |
| `Backward` | 2 | Move backward |
| `Left` | 4 | Move left |
| `Right` | 8 | Move right |
| `Jump` | 16 | Jump |
| `Fire` | 32 | Fire weapon (useless) |
| `Melee` | 64 | Melee attack (useless) |
| `Scope` | 128 | Scope weapon (useless) |

```javascript
// Move forward and jump
bot.movement(Movement.Forward | Movement.Jump);

// Strafe right while moving forward
bot.movement(Movement.Forward | Movement.Right);
```

### Team

Team identifiers for team-based game modes.

| Constant | Value |
|----------|-------|
| `Blue` | 1 |
| `Red` | 2 |

```javascript
// Switch to blue team
await bot.switchTeam(Team.Blue);
```

## WebSocket Close Codes

### CloseCode

WebSocket close codes sent by the game server.

| Code | Name | Description |
|------|------|-------------|
| 4000 | `gameNotFound` | Game session not found |
| 4001 | `gameFull` | Game lobby is full |
| 4002 | `badName` | Invalid username (not used) |
| 4003 | `mainMenu` | Sent by client to return to menu |
| 4004 | `gameIdleExceeded` | Kicked for inactivity |
| 4005-4010 | `corruptedLoginData0-5` | Login data corruption |
| 4011 | `gameMaxPlayersExceeded` | Too many players |
| 4012 | `gameDestroyUser` | User forcefully removed |
| 4013 | `joinGameOutOfOrder` | Invalid join sequence |
| 4014 | `gameShuttingDown` | Server shutting down |
| 4015 | `readyBeforeReady` | Ready packet sent too early |
| 4016 | `booted` | Kicked from game |
| 4017 | `gameErrorOnUserSocket` | Socket error |
| 4018 | `uuidNotFound` | UUID not found |
| 4019 | `sessionNotFound` | Session expired |
| 4020 | `clusterFullCpu` | Server CPU overload |
| 4021 | `clusterFullMem` | Server memory full |
| 4022 | `noClustersAvailable` | No servers available |
| 4023 | `locked` | Game locked |

```javascript
bot.on('disconnect', (code) => {
  if (code === CloseCode.gameIdleExceeded) {
    console.log('Kicked for being AFK');
  } else if (code === CloseCode.sessionNotFound) {
    console.log('Session expired, need to re-login');
  }
});
```
